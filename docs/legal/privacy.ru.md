# Конфиденциальность

Kinotor хранит данные **локально** на вашем компьютере: избранное, история, прогресс, настройки, профили.

Приложение обращается к внешним API (TMDB, Shikimori, трекеры торрентов, Kodik) только для каталога и поиска — по вашим действиям в интерфейсе.

Мы не собираем персональные данные на серверах разработчика через само приложение. Обновления проверяются через GitHub Releases.
