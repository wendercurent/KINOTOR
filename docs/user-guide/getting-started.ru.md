# Первый запуск Kinotor

1. Скачайте установщик с [Releases](https://github.com/wendercurent/KinoTor/releases/latest).
2. Запустите `Kinotor Setup … .exe` (при SmartScreen: «Подробнее» → «Выполнить в любом случае»).
3. Выберите режим аудитории при первом запуске.
4. Откройте карточку фильма → «Торренты» → выберите раздачу → «Смотреть».
