# Частые вопросы

См. полный раздел FAQ в [README.md](../../README.md#частые-вопросы).

Кратко:

- **TorrServer offline** — Настройки → TorrServer → автозапуск; порт 8090.
- **Нет постеров** — проверьте интернет; Настройки → Каталог → источник изображений.
- **Нет торрентов** — откройте карточку тайтла и нажмите «Искать торренты».
- **Антивирус** — сравните файл с [VirusTotal](https://www.virustotal.com/) (ссылка в README).
