## About

**Kinotor** is a free Windows desktop app: TMDB catalog, Shikimori anime, personal library, torrent streaming without full download, and watch party sync.

> This repository ships **installers and docs only**. **Source code is not published.**

Full user guide: [README.md](README.md) (Russian).
