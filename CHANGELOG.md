# Changelog

Все заметные изменения Kinotor документируются в этом файле.

## [0.1.5] — 2026-06-29

### Производительность
- Устранены подвисания из-за лишних полных перерисовок профиля и главной.
- Оптимизирована запись прогресса MPV (throttle, patch вместо полного снимка, отложенный flush при закрытии).
- Плавнее менеджер торрентов и блок «Продолжить просмотр».

### Просмотр
- MPV: названия при автопереходе, плашка «Следующая серия», парсер сезонов/серий (S03E01).
- Встроенный плеер: новый UI, буферизация, scrub tooltip, горячие клавиши, retry, double-tap seek.
- Онлайн Kodik: adblock, кинотеатр-оверлей, выбор источника; исправлен «Смотреть онлайн».
- Трейлеры: исправлено воспроизведение (YouTube Error 153).

### Главная и библиотека
- «Продолжить просмотр»: прокрутка и drag; история — сетка; без дублей в «Популярном».
- «Моя библиотека»: полировка, без дублирующих кнопок; медиатека только через настройки макета.

### Торренты и UI
- Страница торрента: акцент на файлах; менеджер — bulk delete в режиме «Удалить».
- Крупнее окно настроек; убраны quick-actions в сайдбаре.

### Прочее
- AniSkip, автопереход серий, Subtitle Toolkit v2, модалка shortcuts, акценты темы, вкладка «Конфиденциальность».

## [0.1.4] — 2026-06-06

### Рандомайзер «Что посмотреть?»
- Кнопка на странице жанров открывает страницу случайного выбора.
- Компактная панель фильтров: категория, годы, кнопка «Жанры» (всплывающая панель) и «Рандомизировать».
- Одна hero-карточка результата с фоном-backdrop (как на странице тайтла), постер каталожного размера.
- **История рандома** — всплывающий список прошлых подборок (как Episode Browser); кэш постеров и фонов в `cachedTitles`.
- Учитываются безопасный режим, избранное (вес жанров), история просмотров и недавние выпадения.

## [0.1.3] — 2026-06-04

### Главная и медиатека
- Медиатека перенесена в bento-блок **«Популярное»** на главной странице.
- Плитки медиатеки кликабельны: переход в коллекции, избранное, watchlist, статистику и блок «Недавно просмотренные».
- Медиатеку можно свернуть; кнопка разворота в заголовке блока.
- Hero на главной снова на всю ширину сетки.

### Тема и производительность
- Переключение светлой/тёмной темы **без перезагрузки** карточек и постеров в каталоге.
- Оптимизация жанров: один обработчик скролла для flyout, меньше лишних перерисовок DOM, отложенный prefetch только для уже загруженных данных.
- Исправлено сохранение кэша mood-постеров жанров в профиле пользователя (`genreMoodVisuals`).
- TTL кэша TMDB discover увеличен до 24 часов; сессионный кэш discover в renderer.

### Жанры (фильмы / TMDB)
- Bento-сетка жанров v8: mood-карточки, «Быстрый выбор», веер мини-постеров.
- Flyout-постеры вынесены в portal, без обрезки карточкой.
- Размер мини-постеров и flyout приведён к каталогу: **168×252** (2:3, как у `movie-card`).
- Надёжнее подгрузка постеров на bento-карточках: перебор жанров группы, повторное применение кэша, исправлена отсутствующая функция проверки отрисовки.

### Жанры (аниме)
- Ускорена загрузка mood-визуалов; featured-плитки — только выбор жанра (без перехода в тайтл).
- Постеры в «Быстром выборе» и на карточках с корректным fallback зеркал TMDB/Shikimori.

### Статистика
- Переработана страница статистики: bento-плитки, sparkline активности за 12 недель, шаринг постера.
- Данные статистики строятся из durable-источников (`watchActivity`, прогресс, история).

### Карточка фильма и интерфейс
- Hero детальной страницы фильма **без** размытого backdrop (чёткий постер/фон).
- Улучшена кастомная строка заголовка Windows (главное окно и настройки), цвета overlay под тему.

### Настройки
- Уменьшена белая вспышка при открытии: фон окна и тема через preload, критический inline-CSS.
- Убрана фоновая предзагрузка окна настроек (устранены лаги и открытие со второго клика).

### Прочее
- Прокси Cloudflare для VidAPI, guard URL, media mirror, правки TorrServer и watch-party из текущей ветки разработки.
- Документация и конфигурация сборки обновлены под public beta **0.1.3**.
