|------|------------|
| `Kinotor Setup X.Y.Z.exe` | Установщик для Windows (рекомендуется) |
| `Kinotor X.Y.Z.exe` | Портативная версия (без установки) |

**Системные требования:** Windows 10/11, 64-bit, 4 ГБ ОЗУ, ~500 МБ на диске.

## Проверка безопасности

Сборка проверена на VirusTotal:

https://www.virustotal.com/gui/file/SHA256_PLACEHOLDER

> После загрузки `.exe` на VirusTotal замените `SHA256_PLACEHOLDER` на реальную ссылку.
> Добавьте ту же ссылку в `README.md` (раздел «Проверка на VirusTotal»).

## Обновление с предыдущей версии

- Установщик обновит приложение поверх текущей установки.
- Портативная версия: замените старый `.exe` новым; данные профиля хранятся отдельно в `%APPDATA%\kinotor`.
